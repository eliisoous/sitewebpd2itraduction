Exemple d'utilisation du Header Component

Cette page démontre comment utiliser le composant Header réutilisable de PD2i.

Fonctionnalités du composant

  • Navigation responsive (mobile, tablette, desktop)
  • Menus déroulants pour "Coating Equipments" et "Turnkey"
  • Sélecteur de langue multiplateforme
  • Logo personnalisable
  • Bouton Contacts stylisé
  • Configuration entièrement personnalisable

Utilisation simple

// Initialisation basique
const header = new Header();

// Initialisation avec configuration personnalisée
const header = new Header({
    logoSrc: 'mon-logo.png',
    currentLang: 'FR',
    navigationItems: [
        { label: 'Accueil', href: '/', type: 'link' }
    ]
});
